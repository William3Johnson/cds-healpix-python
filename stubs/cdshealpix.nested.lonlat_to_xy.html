cdshealpix.nested.lonlat_to_xy¶

cdshealpix.nested.lonlat_to_xy(lon, lat)¶

Project sky coordinates to the HEALPix space

Parameters
lonastropy.units.Quantity

The longitudes of the sky coordinates.

latastropy.units.Quantity

The latitudes of the sky coordinates.

Returns
x, y: (numpy.ndarray, numpy.ndarray)

The position of the (lon, lat) coordinates in the HEALPix space. \(x \in [0, 8[\) and \(y \in [-2, 2]\)

Examples

>>> from cdshealpix import lonlat_to_xy
>>> import astropy.units as u
>>> import numpy as np
>>> lon = [10, 25] * u.deg
>>> lat = [5, 10] * u.deg
>>> x, y = lonlat_to_xy(lon, lat)
